🤖 Частина 1 — Part 1

Що таке робот? — What is a robot?

Уяви собі: посеред кімнати стоїть робот (robot) — на двох колесах! Він не падає, хоча стоїть прямо, як олівець. Це чудо? Ні — це технологія (technology)!

Робот — це машина, яка вміє виконувати завдання самостійно або за командами людини. Він має датчики (sensors), щоб "відчувати" світ навколо, мотори (motors), щоб рухатись, і комп'ютери (computers), щоб думати!

Наш робот називається ELEGOO Tumbller — і він самобалансуючий (self-balancing). Це означає, що він сам утримує рівновагу, як циркач на канаті. Тільки без канату!

💡 Цікавий факт! Слово "robot" вигадав чеський письменник Карел Чапек у 1920 році. Воно походить від чеського слова "robota" — важка праця. Сьогодні роботи допомагають людям на фабриках, в лікарнях, і навіть у космосі!

🟦 Частина 2 — Part 2

Arduino — мозок рухів

Arduino Nano — це маленька синя плата, менша за твій палець! Але всередині неї є мікросхема — мікроконтролер (microcontroller). Це як крихітний комп'ютер, який виконує одну важливу задачу дуже швидко і точно.

Arduino отримує дані від гіроскопа (gyroscope) та акселерометра — датчиків, які вимірюють, як нахилений робот. Потім Arduino дуже швидко (500 разів на секунду!) обчислює, як потрібно крутити моторами, щоб робот не впав.

Arduino розуміє прості команди (commands): f — вперед (forward), b — назад (backward), l — вліво (left), r — вправо (right), s — стоп (stop).

Arduino_Nano.ino — Serial Monitor
// Програма на мові C++
→ Отримую команду: "f"
⚙ Кут нахилу: +2.3°
⚙ Цільовий кут: 0.0°
→ Мотор Лівий: +180 RPM
→ Мотор Правий: +180 RPM
// RPM = Revolutions Per Minute
// обертів на хвилину
✓ Робот рухається вперед!

Мова програмування Arduino — C++

Arduino програмують мовою C++ (читається "сі-плюс-плюс"). Це дуже швидка мова — ідеальна для мікроконтролерів. C++ is a very fast programming language — perfect for microcontrollers. Програміст пише інструкції, завантажує їх на плату через USB кабель (USB cable), і Arduino їх запам'ятовує назавжди!

Raspberry Pi Zero 2W — це повноцінний маленький комп'ютер! Він менший за кредитну картку, але вміє підключатись до Wi-Fi і запускати справжні програми.

На Raspberry Pi встановлена операційна система Linux і програма на мові Python (Пайтон). Програма запускає веб-сервер (web server) — це як маленький сайт всередині робота!

Коли ти натискаєш кнопку на телефоні через браузер (browser), телефон відправляє команду на Raspberry Pi по Wi-Fi. Raspberry Pi отримує команду і пересилає її до Arduino через кабель USB (USB cable). Arduino крутить моторами. Ось так просто!

IP адреса: 10.0.0.23 — це як "домашня адреса" Raspberry Pi в мережі Wi-Fi. Like a home address on the Wi-Fi network.

⚖️ Частина 4 — Part 4

Як робот балансує? — How does it balance?

Ти коли-небудь намагався балансувати олівець на пальці? Це дуже важко! Але наш робот робить це весь час. Секрет — у гіроскопі (gyroscope) та алгоритмі PID.

Гіроскоп — це датчик, який вимірює, на скільки градусів нахилений робот. It measures the tilt angle in degrees. Він робить це дуже швидко — сотні разів на секунду!

Коли гіроскоп каже "ти нахиляєшся вперед на 3°", Arduino негайно розраховує: "потрібно крутити колеса вперед, щоб зловити рівновагу!" — і робот не падає. Це як ти сам тримаєш рівновагу, але 500 разів на секунду!

Алгоритм PID — що це?

PID — це математичний алгоритм. P = Proportional (пропорційний), I = Integral (інтегральний), D = Derivative (похідний). Він розраховує, наскільки сильно крутити моторами, залежно від кута нахилу. Це як мозок балансування!

balance_loop.ino
// Цикл балансування (500 Гц)
float angle = gyroscope.readAngle();
// angle = поточний кут нахилу
float error = target - angle;
// error = різниця від ідеального
float output = PID.calculate(error);
// output = сила моторів
motors.setSpeed(output);
// Встановити швидкість
Кут: +2.3° → Мотори: +180 RPM
Кут: -1.1° → Мотори: -90 RPM
Кут: 0.0° → Мотори: 0 RPM ✓
